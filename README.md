# Action-
Puzzle 
(epic)
Built **Pixel Raider** — a Game Boy-style top-down shooter with the classic 4-shade green palette:

- Move with arrow keys or the on-screen D-pad, shoot with Space or the A button
- Three enemy types (drones, weavers, brutes) that get faster over time
- Score, lives, hit particles, and a proper GAME OVER / retry screen
- Housed in a chunky console shell so it feels handheld even on desktop

Press **Enter** or **START** to begin. Want me to add power-ups, a boss fight, or level waves next?
